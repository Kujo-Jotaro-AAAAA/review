![](./imgs/pr2.png)

![](./imgs/pr4.png)


2. `clone` 自己的远程仓库

并修改文件

```bash
http://gitlab.psf-dev.com/zhangzhichao/pr-demo.git
cd pr-demo
# 修改README文件后提交至自己的远程仓库
git add .
git commit -m 'change log'
git push
```

3. 提交新的`pr`

![](./imgs/pr5.png)
